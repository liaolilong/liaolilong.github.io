    1.Vue的安装

    利龙 9/21/2021 1.Vue的安装

    # Vue简介

    • 是一套构建用户界面的渐进式框架。

    • 只关注视图层， 采用自底向上增量开发的设计。

    • 目标是通过尽可能简单的 API 实现响应的数据绑定和组合的视图组件。

    特点：

    1.采用组件化模式，提高代码复用率、且让代码更好维护。

    2.声明式编码，让编码人员无需直接操作DOM,提高开发效率。

    3.使用虚拟DOM+优秀的Diff算法，尽量复用DOM节点。

    # 1.学习Vue需要掌握JavaScript基础知识

    ES6语法规范、ES6模块化、包管理器、原型、原型链、数组常用方法、axios、promise

    # 2.Vue_cli脚手架使用

    1.从官网下载安装nodejs (opens new window)

    2.安装cnmp，由于npm会屏蔽一些国外资源，导致npm安装包失败，在命令行中输入：

    npm install -g cnpm –registry=http://registry.npm.taobao.org

    3.安装vue-cli脚手架工具

     npm install -g vue-cli

    4.进入文件夹并初始化，并输入信息，不想填就点回车

    vue init webpack firstVue

    firstVue文件夹名，创建一个该文件名的项目

    5.进入项目文件夹中，运行命令，等待安装所需依赖包

    cnpm install

    6.安装完成之后，会在我们的项目目录firstVue文件夹中多出一个node_modules文件夹，这里边就是我们项目需要的依赖包资源。

    # 3.NPM方法安装vue

    1.安装nodejs

    2.升级npm

    cnpm install npm -g

    3.升级或安装cnpm

    npm install cnpm -g

    4.使用cnpm安装Vue.js

    cnpm install vue

    # 4.直接用script标签引入

    官方文档 (opens new window)

    开发版：vue.js，包含完整警告和调试模式

    生产版本：vue.min.js，删除了警告，33.46KB min+gzip

    • 在项目目录中创建js文件夹将下好的js文件放入

    • 在html文件中引入

      <head>

    <meta charset="UTF-8">


<title>初识Vue</title>

<!--引入Vue-->

    <script type="text/javascript" src="../js/vue.js"></script>

</head>

    # 5.工具安装使用

    1.下载安装Vscode (opens new window)

    2.初始化设置

    （1）中文界面配置

    • 安装中文插件：Chinese (Simplified) Language Pack for Visual Studio Code

    • 右下角弹出重启，点击“yes”

    • 有些机器重启后如果界面没有变化，则 点击 左边栏Manage -> Command Paletet...【Ctrl+Shift+p】

    • 在搜索框中输入“configure display language”，回车

    • 打开locale.json文件，修改文件下的属性 "locale":"zh-cn"

      { 2 // 定义 VS Code 的显示语言。 3 // 请参阅官网 (opens new window)，了解支持的语言列表。 4

      5 "locale":"zh-cn" // 更改将在重新启动 VS Code 之后生效。 6 }

    • 重启vs
