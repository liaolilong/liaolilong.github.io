    # 2.一个Vue实例接管一个容器

    • 一个Vue实例只能接管一个容器是一一对应的关系，根据id进行绑定

      <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>初识Vue</title>
    <!--引入Vue-->
    <script type="text/javascript" src="../js/vue.js"></script>
</head>
<body>
    <!--准备容器-->
    <div id="root">
        <h1>hello,{{name}},{{address}}</h1>
    </div>
    <div id="root1">
        <h1>hello,{{name}},{{address}}</h1>
    </div>

    <script type="text/javascript">
    Vue.config.productionTip = false //设置为 false 以阻止 vue 在启动时生成生产提示。
    
    //创建Vue实例
    new Vue({
        el:'#root',  //el用于指定当前Vue实例为哪个容器服务，值为css选择器字符串。
        data:{ //data中用于存储数据，数据供el所指定的容器去使用。
            name:'Macbook1',
            address:'Macbook2'
        }
    })
    new Vue({
        el:'#root1',  //el用于指定当前Vue实例为哪个容器服务，值为css选择器字符串。
        data:{ //data中用于存储数据，数据供el所指定的容器去使用。
            name:'Macbook1',
            address:'Macbook2'
        }
    })
  
    </script>
</body>
</html>

    # 3.js表达式和js代码（语句）

    1.表达式：一个表达式会生成一个值，可以放在任何一个需要值的地方

    （1）.a

    （2）.a+b

    （3）.demo(1)

    （4）.x === y ? 'a' : 'b'

    2.js代码（语句）

    （1）.if(){}

    （2）.for(){}

    # 4.真实开发环境下

    • 真实开发环境下只有一个Vue实例，并且会配合组件一起使用；

    • 中的xxx要写js表达式，且xxx可以自动读取到data中的所有属性；

    • 一旦data中的数据发生改变，那么模板中用到该数据的地方也会自动更新；
