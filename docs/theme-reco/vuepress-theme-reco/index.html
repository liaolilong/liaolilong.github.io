    个性化主题博客-reco的搭建

    利龙 9/19/2021 个性化主题博客-reco的搭建

    # Vuepress-theme-reco搭建使用方法

    # 简介

    官网链接 (opens new window)

    优秀案例 (opens new window)

    一款简洁而优雅的 vuepress 博客 & 文档 主题

    # 1.环境搭建

    • 安装nodejs (opens new window)

    Node.js 是一个基于 Chrome V8 引擎的 JavaScript 运行环境，有非常强大的npm包管理器。

    • 安装好后打开命令行输入node -v，返回版本号说明搭建成功。

    # 2.创建博客工程

    # 安装vuepress

    npm install vuepress -g

    # 安装脚手架工具

    npm install @vuepress-reco/theme-cli -g

    # 创建项目

    theme-cli init my-blog

    my-blog可替换成自己项目的名字

    # 信息配置

    上述命令执行后会进行一系列的信息配置

    1.输入项目标题

    2.输入对项目的描述

    3.输入作者名字

    4.选择blog选项

    # 依赖包安装

    1.进入项目目录

    cd my-blog

    my-blog替换成填入的项目名

    2.安装依赖包

    npm i

    3.运行测试环境

    npm run dev

    这样就可以本地预览博客了

    该主题是以Vuepress为核心的，文件结构也一致，文件的修改配置可参考目录结构 (opens new window)

    # 3.部署方式

    # github page部署

    1.创建github仓库，仓库名是Owner名加*.github.io*

    Owner名是账号的名字，例如我的github账号名为Kyle，则仓库名应为Kyle.github.io。

    2.填写Description，用于记忆仓库的用途。

    3.点击Creating repository按钮进行创建

    4.回到项目根目录创建deploy.sh脚本用于自动打包项目并推送到github仓库中，如果脚本执行错误可人工按以下脚本进行运行推送

    #!/usr/bin/env sh

# 确保脚本抛出遇到的错误
set -e

# 生成静态文件
npm run build

# 进入生成的文件夹
cd public

# 如果是发布到自定义域名
# echo 'www.example.com' > CNAME

git init
git add -A
git commit -m 'deploy'

# 如果发布到 https://<USERNAME>.github.io
git push -f git@github.com:<USERNAME>/<USERNAME>.github.io.git master

# 如果发布到 https://<USERNAME>.github.io/<REPO>
# git push -f git@github.com:<USERNAME>/<REPO>.git master:gh-pages

cd -

    注意：在GitHub运行时遇到报错可根据百度查找解决方法。

    5.发布成功后在github设置中的page选项可得知网页地址
