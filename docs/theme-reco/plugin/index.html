    theme-reco添加插件

    利龙 9/20/2021 theme-reco添加插件

    # 搭建好博客就可以开始安装有趣的插件了！！！

    # 添加点击星星特效

    插件github链接 (opens new window)

    1.在根目录进入命令行，使用命令进行安装

    yarn add vuepress-plugin-cursor-effects -D
# or use npm
npm i vuepress-plugin-cursor-effects -D

    2.在config.js中进行调用

    module.exports = {
plugins: ['cursor-effects'],
}

    3.可通过选项进行光标特效的配置

    module.exports = {
plugins: [
    [
        'cursor-effects',
        {
            size: 2, // size of the particle, default: 2
            shape: ['star' | 'circle'], // shape of the particle, default: 'star'
            zIndex: 999999999, // z-index property of the canvas, default: 999999999
        },
    ],
],
}

    # 动态标题特效

    插件github链接 (opens new window)

    1.在根目录进入命令行，使用命令进行安装

    yarn add vuepress-plugin-dynamic-title -D
# or use npm
npm i vuepress-plugin-dynamic-title -D

    2.在config.js中进行调用

    module.exports = {
plugins: ['dynamic-title'],
}

    3.可通过选项进行配置

    module.exports = {
plugins: [
    [
        'dynamic-title',
        {
            showIcon: '/favicon.ico',
            showText: '(/≧▽≦/)咦！又好了！',
            hideIcon: '/failure.ico',
            hideText: '(●—●)喔哟，崩溃啦！',
            recoverTime: 2000,
        },
    ],
],
}

    注意：多个插件时使用[]进行调用

    plugins:[
    ['cursor-effects'],
    ['dynamic-title',{
    showIcon: '/favicon.ico',
    showText: '(/≧▽≦/)啦啦啦！',
    hideIcon: '/failure.ico',
    hideText: '(●—●)喔哟！',
    recoverTime: 2000,
    }]
]

    # 在您的 Vuepress 中添加美丽的丝带！

    插件github链接 (opens new window)

    1.在根目录进入命令行，使用命令进行安装

    yarn add vuepress-plugin-ribbon -D
# or use npm
npm i vuepress-plugin-ribbon -D

    2.在config.js中进行调用

    module.exports = {
plugins: ['ribbon'],
}

    3.可通过选项进行配置

    module.exports = {
plugins: [
    [
        'ribbon',
        {
            size: 90, // width of the ribbon, default: 90
            opacity: 0.8, // opacity of the ribbon, default: 0.3
            zIndex: -1, // z-index property of the background, default: -1
        },
    ],
],
}

    # 看板娘动画

    插件github链接 (opens new window)

    1.在根目录进入命令行，使用命令进行安装

    npm install -D vuepress-plugin-helper-live2d

    2.在config.js中进行调用

    module.exports = {
    plugins: ['vuepress-plugin-helper-live2d'],
    }

    3.选择人物

    model:'haru2'

    更多人物id可在github上进行查看

    # 代码行优化

    插件github链接 (opens new window)

    1.在根目录进入命令行，使用命令进行安装

    npm install -D @vuepress-reco/vuepress-plugin-extract-code

    2.在config.js中进行调用

    module.exports = {
plugins: [
    '@vuepress-reco/extract-code'
]
}
